# 📈 Task 4 – Trend Analysis by Release Year

Welcome to Task 4 of my **Data Analysis Using Python Internship** at **Auspify Technologies**.

This project focuses on analyzing how Netflix's content production has evolved over time by examining yearly release trends. The objective is to identify growth patterns, production peaks, and decline periods while presenting meaningful business insights through data visualization and trend analysis.

---

# 🎯 Objective

The goal of this project is to analyze Netflix content production trends over the years by:

- Organizing content by release year
- Calculating yearly content releases
- Identifying growth and decline trends
- Creating professional trend visualizations
- Generating business insights from historical data

# 📈 Project Results

The analysis revealed several important findings:

- 📈 Netflix experienced consistent growth in content production over the years.
- 🚀 The highest growth occurred during the mid-to-late 2010s.
- 📉 Content production slowed after 2020.
- 🎬 Movies consistently outnumbered TV Shows across most release years.
- 🌍 The trend reflects Netflix's rapid global expansion followed by a more selective content strategy.

The visualizations clearly demonstrate how Netflix's content library has evolved over time.
