# 📊 Task 2 – Content Type Analysis Dashboard

Welcome to **Task 2** of my **Data Analysis Using Python Internship** at **Auspify Technologies**.

This project focuses on analyzing the distribution of **Movies** and **TV Shows** available on Netflix using Python. The objective is to gain meaningful insights into Netflix's content library through data visualization and statistical analysis.

---

# 🎯 Objective

The goal of this project is to analyze the cleaned Netflix dataset and understand the distribution of content types by:

* Calculating the total number of Movies and TV Shows
* Comparing content proportions
* Creating informative visualizations
* Generating key business insights from the analysis

# 📈 Project Results

The analysis revealed the following key findings:

* ✅ Total Netflix Titles: **8,790**
* 🎬 Movies: **6,126 (69.7%)**
* 📺 TV Shows: **2,664 (30.3%)**
* 📊 Movie-to-TV Show Ratio: **2.30 : 1**

The visualizations clearly demonstrate that Movies make up the majority of Netflix's content library.
