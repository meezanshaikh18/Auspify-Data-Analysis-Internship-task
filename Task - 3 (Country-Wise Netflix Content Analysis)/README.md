# 🌍 Task 3 – Country-Wise Netflix Content Analysis

Welcome to **Task 3** of my **Data Analysis Using Python Internship** at **Auspify Technologies**.

This project focuses on analyzing the geographical distribution of Netflix content across different countries using Python. The objective is to identify the countries contributing the most content to Netflix's library and present meaningful business insights through data visualization and statistical analysis.

---

# 🎯 Objective

The goal of this project is to analyze Netflix content availability across different countries by:

* Extracting and cleaning country information
* Calculating content count by country
* Identifying the top content-producing countries
* Creating informative visualizations and rankings
* Generating key business insights from the analysis

* Title
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre (Listed In)

This analysis primarily focuses on the **Country** column to explore the geographical distribution of Netflix content.

# 📈 Project Results

The analysis revealed the following key findings:

* 🌎 The **United States** has the highest number of Netflix titles in the dataset.
* 🇮🇳 **India** is among the leading content-producing countries.
* 📊 The Top 10 countries contribute a significant share of Netflix's overall content library.
* 📈 Netflix's content is concentrated in a few major production markets, reflecting strong regional dominance.

The visualizations clearly demonstrate the geographical distribution of Netflix content across the world's leading content-producing countries.
