# 📺 Task 1 – Netflix Data Cleaning & Preparation

Welcome to **Task 1** of my **Data Analysis Using Python Internship** at **Auspify Technologies**.

This project focuses on cleaning and preparing the **Netflix Movies & TV Shows** dataset for business analytics. The objective is to improve data quality by handling missing values, removing duplicates, standardizing data formats, and exporting a clean dataset ready for further analysis.

---

# 🎯 Objective

The goal of this project is to transform a raw Netflix dataset into a clean and reliable dataset suitable for:

* Exploratory Data Analysis (EDA)
* Data Visualization
* Business Intelligence
* Reporting
* Machine Learning preprocessing
